☁️ GitHub 云端同步

通过 GitHub Gist 在多台设备间同步数据。配置一次，之后点按钮即可推送或拉取。

如何获取 Token 和 Gist ID？
  1. 前往 gist.github.com，创建一个新的 Gist（文件名随意，内容随意，选 Create secret gist）
  2. 创建后，从浏览器地址栏复制最后那串字符，即 Gist ID
  3. 前往 GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token
  4. 勾选 gist 权限，生成并复制 Token（以 ghp_ 开头）
  5. 将两者填入下方并保存
